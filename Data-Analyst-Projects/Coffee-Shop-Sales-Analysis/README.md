## Coffee Shop Sales Performance Analysis

This project analyzes coffee shop sales data using Python and Power BI.
The dataset was imported from Excel into a SQL database for storage and access.

### Workflow
- Imported Excel data into SQL
- Used SQL to access and validate the data
- Performed exploratory data analysis (EDA) in Python
- Built interactive Power BI dashboards for sales insights

### Tools Used
- SQL (data storage and access)
- Python (EDA and analysis)
- Power BI (dashboarding)
